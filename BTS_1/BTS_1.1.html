1.A cosa serve bootstrap? 1.È una libreria di HTML, CSS e JavaScript già pronta che ti aiuta a costruire interfacce grafiche
(bottoni, tabelle, menu, griglie, modali...) senza dover scrivere tutto da zero.

2.Cosa devi importare per creare una pagina web con bootstrap 2.Servono per poter scrivere sotto forma di testo i codici nbei tag
ad esempio <p> </p>
Questo invece e' il link che bisognascrivere per poter includere Bootstrap 5 nel file.
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.0/dist/css/bootstrap.min.css">

3.Cos è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito? 3.Un beneficio nell'uso del CDN di Bootstrap 5 è che molti utenti hanno già scaricato la libreria da jsDelivr,
mentre navigano su altri siti. Pertanto, quando visitano il tuo sito, il file può essere caricato direttamente dalla cache,
accelerando i tempi di caricamento. Inoltre, i CDN generalmente assicurano che il file venga fornito dal server più vicino all'utente,
riducendo ulteriormente i tempi di attesa per il caricamento.

4.A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1 4.Bootstrap 5 è progettato per essere responsive, adattandosi automaticamente ai dispositivi mobili. Per garantire un corretto rendering e uno zoom touch,
è necessario aggiungere questo tag meta nell'elemento head.
la parte `width=device-width` imposta la larghezza della pagina in base alla larghezza dello schermo del dispositivo, mentre `initial-scale=1` imposta lo zoom iniziale quando la pagina viene caricata.
